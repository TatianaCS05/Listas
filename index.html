Lista enlazada

Una lista enlazada simple es una estructura de datos donde cada nodo contiene un valor y una referencia al siguiente nodo. Permite inserciones y eliminaciones eficientes, pero solo se puede recorrer en una dirección. Es útil para colecciones dinámicas de elementos.

Lista enlazada doble

La lista enlazada doble tiene nodos que contienen referencias tanto al siguiente como al anterior. Esto permite recorrer la lista en ambas direcciones, facilitando inserciones y eliminaciones desde cualquier punto. Es ideal para aplicaciones que requieren acceso bidireccional.

Lista enlazada circular

En una lista enlazada circular, el último nodo apunta de nuevo al primero, formando un ciclo. Esto permite recorrer la lista indefinidamente. Es útil en aplicaciones que necesitan un acceso cíclico a los elementos, como en juegos o rondas de turnos.
